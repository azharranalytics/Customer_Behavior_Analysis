# Customer_Behavior_Analysis
Customer Shopping Behavior Analysis project that explores 3,900 transactions to uncover insights on spending patterns, customer segments, and product preferences. Data was cleaned and analyzed using Python (Pandas) and SQL (PostgreSQL), with results visualized in an interactive Power BI dashboard to support data-driven business decisions.

Overview:

This project analyzes customer shopping behavior using transactional data to identify spending patterns, customer segments, and product preferences. The insights help businesses make data-driven decisions to improve marketing strategies, customer retention, and revenue growth.

Dataset:

Records: 3,900 transactions
Features: Customer demographics, purchase details, discounts, shipping type, subscription status, and product ratings
Data Processing: Missing values handled and features standardized for analysis
Tools & Technologies
Python (Pandas, NumPy) – Data cleaning & Exploratory Data Analysis (EDA)
SQL (PostgreSQL) – Business query analysis
Power BI – Interactive dashboard visualization
Gamma / PowerPoint – Presentation of insights

Project Steps:

Loaded dataset using Python
Performed data cleaning and preprocessing
Conducted Exploratory Data Analysis (EDA)
Stored processed data in SQL database
Ran SQL queries to answer business questions
Built interactive dashboard in Power BI
Created report and presentation
Dashboard

The Power BI dashboard provides visual insights into:

Revenue distribution
Customer segmentation
Top-performing products
Purchase trends
Subscription behavior
Results
Identified high-value customer segments
Found top-rated and most purchased products
Analyzed impact of discounts on revenue
Compared subscriber vs non-subscriber spending patterns
Provided business recommendations for marketing and retention
How to Run
Download the dataset
Run Python scripts for cleaning and EDA
Load processed data into PostgreSQL
Execute SQL queries
